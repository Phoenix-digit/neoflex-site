# Neoflex Store — Phoenix Budget Control

Site statique prêt pour GitHub Pages.

## Fichiers
- `index.html`
- `style.css`
- `script.js`
- `assets/logo.png`

## Mise en ligne rapide sur GitHub Pages
1. Crée un nouveau dépôt GitHub.
2. Charge tous les fichiers de ce dossier.
3. Va dans **Settings > Pages**.
4. Choisis la branche principale et le dossier `/root`.
5. Publie.

## Personnalisation immédiate
- Remplace les liens WhatsApp dans `index.html`.
- Ajoute ton lien Instagram.
- Ajuste les prix et messages.
- Tu peux aussi passer ton numéro WhatsApp dans l'URL du site :
  `?wa=228XXXXXXXX`
